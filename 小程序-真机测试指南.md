# 止观AI 小程序 - 真机测试指南

## 📋 测试前准备

### 1. 需要的信息
- [ ] 微信小程序 AppID（从微信公众平台获取）
- [ ] 微信开发者工具（已安装）
- [ ] 测试用手机（已安装微信）

### 2. 配置 AppID

打开 `project.config.json`，替换 placeholder：

```json
{
  "appid": "wx_appid_placeholder",  // ← 替换为你的 AppID
  ...
}
```


## 🚀 真机测试步骤

### 第一步：导入项目

1. 打开 **微信开发者工具**
2. 点击「导入项目」
3. 选择 `zhiguan-miniprogram` 文件夹
4. 输入你的 AppID
5. 点击「导入」

### 第二步：开发者工具预览

1. 等待编译完成
2. 检查控制台是否有报错
3. 在模拟器中测试基本流程：
   - 首页 → 选择时长 → 开始练习
   - 练习页 → 倒计时 → 完成评分
   - 统计页 → 查看记录

### 第三步：真机预览

1. 点击开发者工具右上角的「预览」按钮
2. 扫描二维码
3. 在手机上测试

### 第四步：真机调试

1. 点击「真机调试」按钮
2. 扫描二维码
3. 可以在开发者工具查看手机端的 console 日志

## 🔧 常见问题

### Q: 提示 "appid 不匹配"
**A:** 检查 `project.config.json` 中的 appid 是否与微信公众平台的一致。

### Q: tabBar 图标不显示
**A:** 图标已生成在 `images/` 目录，如仍不显示，检查图片路径是否正确。

### Q: 练习记录无法保存
**A:** 目前使用本地存储，真机测试时数据会保存在微信本地。

### Q: 音频无法播放
**A:** 引导模式音频暂未接入，需要上传音频文件到服务器或小程序本地资源。
