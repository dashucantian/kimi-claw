# 止观AI 微信小程序

一款专注于正念冥想的微信小程序，帮助用户在繁忙的生活中找到内心的宁静。

## 功能特性

- 🧘 **止观练习**：支持 3/5/10/20 分钟多种时长选择
- 🔔 **音效提示**：大磬、引磬、木鱼、风铃、水滴等多种音效
- 📊 **修行记录**：日历热力图、连续天数统计
- 🌙 **沉浸体验**：呼吸动画、深色模式

## 页面结构

```
pages/
├── index/          # 首页 - 练习入口
├── practice/       # 练习页 - 计时器+呼吸动画
├── record/         # 记录页 - 日历+统计
└── profile/        # 我的页 - 设置
```

## 技术栈

- 微信小程序原生框架
- WXSS 样式
- 本地存储 (Storage)

## 开发说明

1. 使用微信开发者工具打开项目
2. 填写自己的 appid
3. 添加 tabBar 图标到 images 目录
