# 止观AI - 引导音频制作指南

## 制作方案

### 方案A：AI语音合成（推荐，快速）
使用ElevenLabs/Azure TTS等工具，选择柔和、沉稳的声音。

### 方案B：真人录制（最佳效果）
由修行者或专业配音录制，更有感染力。

### 方案C：混合方案
AI生成初版，真人优化关键段落。
