# 止观AI 小程序设计方案

## 一、设计系统

### 色彩体系
- **主色**：#1a1a2e（深夜蓝）→ #16213e（深海蓝）渐变
- **强调色**：#4a9eff（宁静蓝）
- **文字**：#ffffff（主文字）、rgba(255,255,255,0.6)（次要文字）
- **背景层级**：
  - 底层：渐变背景
  - 中层：卡片 rgba(255,255,255,0.05)
  - 前景：纯白文字

### 字体
- 主字体：-apple-system, PingFang SC, Microsoft YaHei
- 倒计时：DIN Alternate / SF Mono（等宽数字）

### 间距
- 页面边距：20px
- 卡片间距：16px
- 按钮高度：48px（标准）、160px（主按钮圆形）

## 三、首页（练习页）详细设计

### 布局结构
```
┌─────────────────────────┐  ← 安全区顶部
│                         │
│    早安                 │  ← 问候语 28px
│    每日止观，回归当下    │  ← 副标题 14px
│                         │
├─────────────────────────┤
│  ┌─────┐ ┌─────┐       │
│  │  0  │ │  0  │       │  ← 统计卡片
│  │今日 │ │连续 │       │
│  └─────┘ └─────┘       │
├─────────────────────────┤
│                         │
│         ○○○○○          │  ← 呼吸指示点
│                         │
│      ┌─────────┐        │
│      │         │        │
│      │  开始   │        │  ← 主按钮 160px
│      │  止观   │        │     渐变蓝
│      │         │        │
│      └─────────┘        │
│                         │
│      3分钟 · 引导       │  ← 当前设置
│                         │
├─────────────────────────┤
│  3分  5分  10分  20分   │  ← 时长选择
│  [选中状态：蓝底白字]    │     横向滑动
├─────────────────────────┤
│  引导  │  自主           │  ← 模式切换
│  [选中状态：蓝底]        │     分段控制器
└─────────────────────────┘  ← 底部导航
```

### 交互说明
1. **主按钮**：
   - 默认：渐变蓝 + 呼吸动画（scale 1.0 → 1.02，3秒循环）
   - 按下：scale 0.95
   - 点击：跳转到练习页

2. **时长选择**：
   - 横向滚动，选中项放大1.1倍 + 蓝色背景
   - 滑动时轻微震动反馈

3. **模式切换**：
   - 滑动切换，选中项背景色变化

## 四、练习页详细设计

### 布局结构
```
┌─────────────────────────┐
│ ← 返回        3分钟止观  │  ← 顶部导航
│               引导练习   │
├─────────────────────────┤
│                         │
│                         │
│      ╭───────────╮      │
│      │           │      │
│      │  03:00    │      │  ← 倒计时 72px
│      │           │      │     居中
│      ╰───────────╯      │
│                         │
│       调息              │  ← 阶段标签
│                         │
│  吸气...呼气...         │  ← 引导文字
│  保持自然的呼吸节奏      │
│                         │
│      ○ ○ ○ ○ ○         │  ← 进度指示
│                         │
├─────────────────────────┤
│  ┌─────┐ ┌─────┐       │
│  │暂停 │ │结束 │       │  ← 控制按钮
│  └─────┘ └─────┘       │
└─────────────────────────┘
```

### 背景效果
- 渐变：#1a1a2e → #16213e（180度）
- 呼吸光晕：中央 radial-gradient，随呼吸周期缩放

### 倒计时动画
- 数字：等宽字体，72px，字重200
- 呼吸同步：4秒周期（吸气2秒 → 呼气2秒）
- 视觉提示：外圈圆环随呼吸缩放

### 阶段提示
| 进度 | 阶段 | 提示文字 |
|------|------|----------|
| 0-17% | 调身 | 让身体自然放松，脊柱轻轻挺直 |
| 17-67% | 调息 | 吸气...呼气...保持自然的呼吸节奏 |
| 67-89% | 观心 | 觉察身体的感觉，不评判，只是知道 |
| 89-100% | 收功 | 慢慢把觉知带回，准备结束练习 |

### 音效提示
- 阶段切换时播放对应音效
- 可点击右下角图标切换静音

## 七、交互动效规范

### 1. 页面转场
- 类型：滑动（slide）
- 时长：300ms
- 缓动：ease-out

### 2. 按钮反馈
- 按下：scale 0.95，100ms
- 释放：scale 1.0，200ms
- 主按钮：持续呼吸动画（scale 1.0 → 1.02，3秒循环）

### 3. 呼吸动画
- 周期：4秒（吸气2秒 → 呼气2秒）
- 元素：
  - 外圈圆环：scale 1.0 → 1.1
  - 光晕：opacity 0.3 → 0.5
- 缓动：ease-in-out

### 4. 数字变化
- 倒计时：无动画（保持清晰）
- 统计数字：countUp动画，1秒
