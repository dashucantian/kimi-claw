# 止观AI - 项目完整文档

## 📱 项目概述

**止观AI** — 一个极简的冥想/正念练习工具，基于传统「止观」修行方法，帮助用户在3-10分钟内快速进入专注状态。

### 核心理念
- **止** — 让心停下来，不再散乱
- **观** — 觉察当下，不评判
- **极简** — 无社交、无压力、无学习成本

---

## 🎯 功能规格

### 1. 首页
- 时间问候（早上好/下午好/晚上好）
- 今日/连续/总时长统计
- 时长选择：3分钟 / 5分钟 / 10分钟
- 模式选择：引导模式 / 自主模式
- 场景提示（通勤/休息/睡前/焦虑）

### 2. 练习页
- 倒计时显示（MM:SS格式）
- 呼吸动画背景（脉动圆环）
- 阶段引导文字：
  - 调身（0-17%）— 身体放松
  - 调息（17-67%）— 呼吸自然
  - 观心（67-89%）— 觉察不评判
  - 收功（89-100%）— 带回觉知
- 随机提示语（每30秒更换）
- 进度条
- 暂停/继续/结束控制

### 3. 完成页
- 练习时长统计
- 1-5星评分
- 备注输入
- 保存记录

### 4. 统计页
- 总览：总次数 / 总小时 / 连续天数
- 本周日历（练习标记）
- 历史记录列表（时间/时长/评分/备注）

## 📝 音频脚本

### 3分钟通勤版
```
【调身】30秒
找个地方坐下，脊柱自然挺直，不需要刻意用力。
肩膀放松，双手自然放在腿上。

【调息】90秒
现在，把注意力放在呼吸上。
不需要控制呼吸，只是知道它在进出。
吸气...呼气...
如果心跑掉了，轻轻拉回来就好，不要责怪自己。

【观心】30秒
觉察身体的感觉，不评判，只是知道。
念头来了，让它来；念头走了，让它走。
你只是在看。

【收功】30秒
慢慢把觉知带回，动一动手指，动一动脚趾。
当你准备好了，轻轻睁开眼睛。
这一坐，功德圆满。
```

### 5分钟日常版
```
【调身】50秒
找一个舒适的坐姿，可以是椅子，也可以是垫子。
脊柱轻轻挺直，像一串珠子被 gently 提起来。
肩膀放松，让它们沉下去。
双手自然安放，舌头轻抵上颚。
闭上眼睛，或者让视线柔和地垂下。

【调息】150秒
把注意力带到呼吸上。
不需要改变呼吸的节奏，只是观察它本来的样子。
吸气时，知道自己在吸气。
呼气时，知道自己在呼气。
...
心会跑，这是正常的。
发现心跑掉了，轻轻把它带回来。
不带责备，不带评判。
就像训练一只小狗，耐心而温和。

【观心】50秒
现在，扩大你的觉察。
不只是呼吸，还有身体的感觉。
也许是接触点的压力，也许是温度，也许是微小的颤动。
不追逐，不排斥。
只是知道。

【收功】30秒
慢慢准备结束。
深深吸一口气，缓缓呼出。
动一动手指，动一动脚趾。
把觉知带回整个身体，带回这个空间。
当你准备好了，轻轻睁开眼睛。
愿这份宁静，延续到接下来的时刻。
```

### 10分钟深入版
